#  B-labs Musix 

### Current Features
- Loads multiple files in one microsecond
- 10 Band Equalizer
- Next track widget
- Online lyrics
- Streaming music ( UG Music currently )
- Visualisers
- Room Effects (Scene , Small Room ,...)

* More features are coming....
